================================================================================
VULNÉRABILITÉS CRITIQUES (Corriger immédiatement)
================================================================================

1. LIMIT INJECTION - polyvalence_logger.py (2 occurrences)
   Fichier: App/core/services/polyvalence_logger.py
   Lignes: 251 et 284
   Pattern: query += f" LIMIT {int(limit)}"
   Risque: Une valeur limit mal validée pourrait causer une injection SQL
   Impact: Accès à toutes les données ou modification de logique de requête
   Correction estimée: 30 minutes

2. UPDATE SET DYNAMIQUE - document_service.py
   Fichier: App/core/services/document_service.py
   Ligne: 396
   Pattern: sql = f"UPDATE documents SET {', '.join(updates)} WHERE id = %s"
   Risque: Les noms de colonnes sont actuellement sûrs mais le pattern est fragile
   Impact: Risque augmente si code refactorisé
   Correction estimée: 45 minutes

3. TABLE NAME INJECTION - gestion_documentaire.py
   Fichier: App/core/gui/gestion_documentaire.py
   Lignes: 254-261
   Pattern: query = f"""FROM {table_name}""" où table_name = 'personnel'
   Risque: Actuellement sans danger mais pattern dangereux
   Impact: Risque si table_name devient configurable
   Correction estimée: 15 minutes

================================================================================
VULNÉRABILITÉS MOYENNES (Corriger dans la prochaine version)
================================================================================

1. WHERE CLAUSE DYNAMIQUE - historique.py
   Fichier: App/core/gui/historique.py
   Ligne: 674
   Pattern: f"WHERE {' AND '.join(where)}"
   Risque: search_text n'est pas validé, risque de manipulation LIKE
   Impact: Limite faible, les paramètres sont séparés correctement
   Correction estimée: 30 minutes

2. CONDITIONAL SQL - contrat_service.py
   Fichier: App/core/services/contrat_service.py
   Lignes: 320-322
   Pattern: query += " AND c.actif = 1"
   Risque: Anti-pattern de construction dynamique
   Impact: Peut devenir dangereux si plus de conditions dynamiques ajoutées
   Correction estimée: 20 minutes
