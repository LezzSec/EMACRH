│                                                                        │
│  ERREUR: "Structure du projet incorrecte"                              │
│  → Vous devez être dans build-scripts/ pour exécuter le script        │
│    Solution: cd build-scripts && build_optimized.bat                  │
│                                                                        │
│  ERREUR: "Python n'est pas installé"                                  │
│  → Python n'est pas dans le PATH                                      │
│    Solution: Installer Python 3.8+ et l'ajouter au PATH               │
│                                                                        │
│  ERREUR: "PyInstaller failed"                                         │
│  → Dépendances manquantes                                             │
│    Solution: cd ../App && pip install -r requirements.txt             │
│                                                                        │
│  BUILD TROP VOLUMINEUX (>100 MB)                                      │
│  → UPX n'est pas installé                                             │
│    Solution: Installer UPX depuis https://upx.github.io/              │
│                                                                        │
│  L'EXE NE DÉMARRE PAS                                                 │
│  → Tester en mode console pour voir les erreurs                       │
│    Solution: Dans EMAC_optimized.spec, mettre console=True            │
